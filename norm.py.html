Normalization



  • * FASTA/FASTQ file with sequences of nucleotides and aminoacids.


  • * If you want to use a previous output file, you should insert the ID of the file.


  • * Desired coverage of the sequences (30x recommended).


  • * Word size to use for normalization.


  • * We will send you an email when your process is finished.
Inputs:
FASTa/FASTQ file, a coverage and a K-mer.
Process:
This step will reduce their sequences to a given coverage so that when assembling this work will be less computationally intensive (takes less time).
Output:
FASTA/FASTQ normalized file.
